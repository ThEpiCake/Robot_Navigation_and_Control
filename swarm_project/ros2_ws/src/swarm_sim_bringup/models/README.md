# Simulation Models (not tracked)

The Gazebo models used by `swarm_sim_bringup` (`x500_base`, `OakD-Lite`, `Lake_House`, etc.)
are standard PX4 SITL / Gazebo Fuel assets (~277MB) and are not tracked in this repository.

They are installed automatically with [PX4-Autopilot](https://github.com/PX4/PX4-Autopilot)
(`Tools/simulation/gz`) or can be downloaded from [Gazebo Fuel](https://app.gazebosim.org/fuel).
The package also depends on [px4_msgs](https://github.com/PX4/px4_msgs) (clone into `src/`).
